# Empyreal AI Community Builder - Flutter

Flutter application for building and managing AI communities.

## Getting Started

### Prerequisites

- Flutter SDK (3.10.3 or higher)
- Dart SDK
- Android Studio / Xcode (for mobile development)
- VS Code or Android Studio (recommended IDEs)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```
